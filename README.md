# Student Result Analyzer (C++)

A menu-driven console application built in C++ to manage and analyze student academic results.

## Features
- Add student details
- Calculate total, percentage, and grade
- Display class analysis
- Search student by roll number
- Find class topper
- Save results to a file

## Technologies Used
- C++
- STL (vector, algorithm)
- File Handling (fstream)
- VS Code

## How to Run
```bash
g++ main.cpp -o result
./result
